# Comparison Report Folder

This folder is organized into three parts:

## Latest Generated Artifacts

These files are overwritten by the default `run_comparison.py` flow:

- `comparison_results.txt`
- `comparison_data.json`
- `dashboard.html`
- latest run logs

## `reports/`

Final report drafts and research write-ups:

- `report.md`
- `final_combined_research_report.md`

## `presentations/`

Presentation and viva scripts:

- `presentation_script.md`
- `full_project_presentation_script.md`

## `runs/`

Saved per-dataset evidence bundles, such as:

- `runs/facebook-combined/`
- `runs/twitter-combined/`

These preserve proof files, logs, and dashboard data for specific datasets.
